What Are Embeddings?

The secret sauce that makes semantic search possible.

ğŸ’¡ Simple Definition

An embeddingEmbedding

A list of numbers (typically 384-1024) that represents the meaning of text. Similar meanings produce similar numbers, enabling semantic search. is a way to convert text into numbers that a computer can understand and compare.

ğŸ—ºï¸ Analogy: GPS Coordinates for Words

Think of embeddings like GPS coordinates, but for words:

Physical World

New York: (40.7, -74.0)
Los Angeles: (34.0, -118.2)
Paris: (48.8, 2.3)

NYC & LA are closer (both in USA)

Word World

"happy": (0.8, 0.6, 0.2...)
"joyful": (0.79, 0.61, 0.19...)
"sad": (-0.7, -0.5, 0.1...)

"happy" & "joyful" are closer (similar meaning)

How Embeddings Are Created
1

Input Text

"responsive navbar"
2

Tokenization

["respon", "sive", "nav", "bar"]
3

Neural Network Processing

Millions of mathematical transformations

4

Output Vector

[0.234, -0.567, 0.891, 0.123, ...]

Typically 384 to 1024 numbers

âœ… Solution: Embeddings capture meaning as numbers

def create_search_embedding(snippet):
    """Convert search_text to numbers."""

Step 4: Store in ChromaDB 
def store_in_chromadb(snippets, embeddings):
    """Store everything in ChromaDB."""
    
    # Create client with persistent storage
    client = chromadb.PersistentClient(path=CHROMA_PERSIST_DIR)
    
    # Create collection with cosine similarity
    collection = client.create_collection(
        name=COLLECTION_NAME,
        metadata={"hnsw:space": "cosine"}
    )
    
    # Add all snippets
    collection.add(
        ids=[s.id for s in snippets],
        embeddings=embeddings,
        documents=[s.code for s in snippets],  # Store CODE!
        metadatas=[{"category": s.category} for s in snippets]
    )
    
    return collection

1. RETRIEVE: Search for Similar Snippets 
def search_snippets(collection, query):
    """Find code snippets similar to the query."""
    
    # ChromaDB auto-embeds the query and searches
    results = collection.query(
        query_texts=[query],  # "I need a navbar"
        n_results=3,           # Get top 3 matches
        include=["documents", "distances"]
    )
    
    snippets = []
    for i in range(len(results['ids'][0])):
        distance = results['distances'][0][i]
        similarity = 1 - distance  # Convert to 0-1
        
        if similarity >= 0.5:  # Only good matches
            snippets.append({
                'code': results['documents'][0][i],
                'similarity': similarity
            })
    
    return snippets
2. AUGMENT: Build the RAG Prompt 
def build_rag_prompt(query, snippets):
    """Build prompt with retrieved examples."""
    
    prompt = """You are a code assistant.
Use these examples as reference:

"""
    
    # Add each retrieved snippet
    for i, snippet in enumerate(snippets):
        prompt += f"""
--- Example {i+1} ---
{snippet['code']}
"""
    
    # Add the user's request
    prompt += f"""
--- USER REQUEST ---
{query}

Generate code following the patterns above.
"""
    
    return prompt
3. GENERATE: Create New Code 
def generate_code(prompt):
    """Send to Ollama and get generated code."""
    
    response = ollama.chat(
        model='codellama',
        messages=[{
            'role': 'user',
            'content': prompt
        }]
    )
    
    return response['message']['content']


# Complete pipeline:
def answer_question(collection, query):
    snippets = search_snippets(collection, query)  # 1. RETRIEVE
    prompt = build_rag_prompt(query, snippets)      # 2. AUGMENT
    code = generate_code(prompt)                   # 3. GENERATE
    return code
